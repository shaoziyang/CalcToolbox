# CalcToolbox

A small cross platform software for engineer calculation.

## feature

- Multiple CRC calculations, with predefined patterns
  - CRC4
  - CRC5
  - CRC6
  - CRC7
  - CRC8
  - CRC16
  - CRC32
- Multiple base conversion
- Multiple types of numbers to byte buffer conversion
- Constants display
- more
